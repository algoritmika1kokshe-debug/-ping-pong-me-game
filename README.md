# -ping-pong-me-game
Здравстувуйте это простькая игра написаная начинающим програмистом так что не судите строго
игра предстовляет себя обычную классическую игру пинг понг где играют два игрока с клавиатуры за две вертикальные платформы и они не должны добустить что бы шарпик костнулся стены за платформой

Управление:
      для первого игрока- W  движение ракетки в верх, s- движене в низ
      для второго игрока - ↑ движение в верх, ↓- движение в низ
